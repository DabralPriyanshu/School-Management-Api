School Management API v1.0

POST /api/v1/schools/addSchool

Adds a new school to the system. Data is validated before insertion.

Required JSON Body: 
{
  "name": "Example School",
  "address": "123 Street, City",
  "latitude": 12.3456,
  "longitude": 78.9012
}
Responses:
  • 201: School successfully created.
  • 400: Bad Request (Validation failed).
  • 500: Internal Server Error.

GET /api/v1/schools/listSchools

Retrieves schools. If latitude and longitude are provided in query params, results are sorted by proximity.

Query Parameters: ?latitude=12.34&longitude=78.90

Responses:
  • 200: List of schools (sorted if coords provided).
  • 500: Internal Server Error.
